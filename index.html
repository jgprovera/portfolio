Organizador de archivos en Python

Herramienta de automatización desarrollada en Python para la clasificación y organización de archivos según su extensión. El programa inspecciona el contenido de un directorio, identifica el tipo de cada archivo y ejecuta su reubicación automática dentro de una estructura de carpetas generada dinámicamente.

La implementación incorpora funciones modulares, estructuras de datos basadas en diccionarios, condicionales, bucles, validación de rutas y operaciones sobre el sistema de archivos mediante los módulos estándar os y shutil.

Avances: desarrollo de una estructura modular con funciones específicas, clasificación automática por extensión, creación dinámica de carpetas, validación básica de rutas y publicación del proyecto como repositorio público documentado.

Tecnologías: Python · Linux (Arch Linux) · Git · GitHub
