Organizador de archivos en Python

Herramienta desarrollada para automatizar la clasificación y organización de archivos según su extensión.

El proyecto implementa lógica de automatización aplicada a la gestión de directorios y archivos, utilizando estructuras de control, funciones y operaciones sobre el sistema de archivos para resolver una necesidad práctica de organización.
